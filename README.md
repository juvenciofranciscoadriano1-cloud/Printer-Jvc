# Meu Site

Este é um projeto simples de site web estático.

## Estrutura

- `index.html`: Página principal do site.
- `style.css`: Estilos CSS para a aparência.
- `script.js`: JavaScript para interatividade.

## Como funciona

1. **HTML**: Define a estrutura da página (cabeçalho, corpo, rodapé).
2. **CSS**: Controla o visual e layout.
3. **JavaScript**: Adiciona funcionalidades dinâmicas, como eventos de clique.

## Como executar

Abra o arquivo `index.html` em um navegador web.

## Uso

- Clique no botão para ver uma mensagem.

## Solução de problemas

- Certifique-se de que todos os arquivos estão na mesma pasta.
- Use um navegador moderno para melhor compatibilidade.
